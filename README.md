# AI Security Data Platform

## Overview

The AI Security Data Platform is an embedded system designed to collect, organize, and analyze sensor information from security devices and robotic platforms. The project provides a centralized framework for storing camera, motion, and environmental data to support future AI-assisted monitoring, event detection, and autonomous decision making.

## Features

- Sensor data collection
- Camera event logging
- Motion detection records
- AI-ready datasets
- Data visualization
- Expandable architecture

## Technologies

- Python
- NVIDIA Jetson
- SQLite / PostgreSQL
- OpenCV
- Computer Vision
- Embedded Systems
